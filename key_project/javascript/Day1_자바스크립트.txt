자바스크립트 (JavaScriopt)

프로그램(program)이란?
프로그램은 컴퓨터가 실행할 수 있는 명령들로 이루어집니다. 컴퓨터 프로그래밍에서 컴퓨터가 실행할 수 있는 명력들을 실행문이라고 합니다. 즉, 프로그램은 특정 결과를 얻기 위해서 컴퓨터에 의해 실행되는 실행문들을 만들어 사용하는 것입니다.


자바스크립트란?
1. 웹의 동작을 구현하기 위한 언어
2. 웹 브라우저에서 사용
3. 웹 브라우저에 자바스크립트 인터프리터가 내장
4. 객체 기반의 스크립트 언어
5. HTML 문서 내에서 <script>~</script>안에 작성 (위치는 상관 없음)
6. 대소문자를 구별한다.


자바스크립트의 출력 방법
1. window.alert() 메소드에 의한 출력
2. document.write() 메소드에 의한 <body>에 출력
3. console.log() 메소드에 의한 브라우저  console 창에 출력
4. HTML DOM 요소를 이용한 inner.HTML 프로펕로 <body>에 출력
프로퍼티 : 값을 저장한는 역할


자바스크립트의 주석
// : 한줄 주석문
/**/ : 여러줄 주석문


자바스크립트 적용방법
1. 내부 자바스크립트
<script> ~ </script>
2. 외부 자바스크립트 파일로 적용
<script src="자바스크립트 파일 경로"></script>


변수(Variable)
변수란 데이터를 저장할 수 있는 메모리 공간을 의미합니다.
그 값은 변경될 수 있습니다. 자바스크립트에서는 var(let) 키워드를 사용하여 변수를 선언하고 선언되지 않은 변수를 사용하려고 하거나 접근하려고 하면 오류가 발생합니다.

var 변수이름;

* 변수이름 작성법
영문, 숫자, 특수문자를 사용하여 만들 수 있습니다. (단, 이름은 반드시 영문으로 시작해야 하며, 
특수문자는 _만 사용하여 만드는 것이 좋습니다. 예약어를 사용할 수 없습니다.)

예)
var num;
num = 10;
document.write(num);


자바스크립트 데이터 타입
타입이란 프로그램에서 다룰 수 있는 값의 종류를 의미합니다. 자바스크립트에서 여러가지 형태의
타입을 미리 정의하여 제공하고 있습니다.

1. 숫자형(numbedr)
자바스크립트는 다른 언어와 달리 정수와 실수를 따로 구분하지 않습니다. 모든 수를 실수 하나로만 표현합니다.

2. 문자열형(string)
자바스크립트에서는 문자열을 쌍따옴표나 따옴표로 둘러싸인 문자의 집합을 의미합니다.
쌍따옴표는 작은 따옴표로 둘러싸인 문자열에만 포함될 수 있고, 따옴표는 쌍따옴표로 둘러싸인 문자열에만 포함될 수 있습니다.

str1 = "'강은영 : 여러분 반가워요'";
str2 = '"강은영 : 여러분 반가워요"';

3. 불리언형 (boolean)
불리언의 값은 true(참), false(거짓)으로 표현합니다.
자바스크립트에서 불리언 값은 예약어인 true, false를 사용하여 나타낼 수 있습니다.

var b1 = true;  // 1
var b2 = false; // 0

4. undefined, null형
자바스크립트에서 null이란 object 타입이며, 아직 "값"이 정해지지 않은 것을 의미합니다. 또한
undefined란 null과는 달리 "타입"이 정해지지 않은 것을 의미합니다. 따라서 자바스크립트에서는  
undefined는 초기화 되지 않은 변수나 존재하지 않은 값에 접근할 때 반환됩니다.

5. 객체형(object)
자바스크립트의 기본 타입입니다. 객체란 실생활에서 우리가 인식할 수 있는 사물을 프로그래밍에서
메모리에 저장하는 형태를 말합니다. 객체는 여러 프로퍼티나 메소드를 같은 이름으로 묶어놓은 일종의 집합체입니다. 

자바스크립트 타입 변환
자바스크립트의 변수는 타입이 고정되어 있지 않으며, 같은 변수에 다른 타입의 값을 대입할 수도 있습니다. 
var a = 10;      // number
var a = "10";    // number -> string

* NaN : Not a Number의 축약형으로, "정의되지 않은 값"이나 "표현할 수 없는 값"이라는 의미입니다.
NaN은 Number 타입의 값으로 0을 0으로 나누거나, 숫자로 변환할 수 없는 연산을 시도하는 경우 반환됩니다.

1. 묵시적 타입 변환
자바스크립트는 특정 타입의 값을 기대하는 곳에 다른 타입의 값이 오면, 자동으로 타입을 변환해서 사용합니다.
 var a = 10;
var b = 20;
var c = "문자";
var d = a + b;   // 1020문자열 연결
d = a - b         // -10 묵시적 타입 변환
d = a - c         //NaN

2. 명시적 타입 변환
자바스크립트는 자동으로 묵시적 타입 변환을 사용하지만, 명시적으로 타입을 변환할 방법도 제공합니다.
Number() : 문자를 숫자로 변환
String() : 숫자나, 불린등의 자료를 문자형으로 변환
Boolean() : 문자나 숫자등의 자료를 불린형으로 변환
Object() :  모든 자료형을 객체형으로 변환
parseInt() :  문자를  int형으로 변환합니다. (정수)
parseFloat() : 문자를 float형으로 변환합니다. (소수)

자바스크립트의 대화상자
window.alert() : 사용자에게 간단한 메세지를 보여주고, 사용자의 확인을 기다립니다.
window.confirm() : 사용자에게 간단한 메세지를 보여주고, 확인이나 취소를 누르면 그 결과를 boolean값으로 반환합니다.
window.prompt() :  사용자에게 간단한 메세지를 보여주고, 사용자가 입력한 문자열을 반환합니다.
